Protein Classifier

Deep learning has been gaining a lot of popularity as many scientists and engineers are using these models to solve and optimize problems. On of the most important appliations involves classifying protein sequence data. This project focuses on developing a protein sequence classification model in which the sequential data's family is known using Pfam. Using tensorflow, Keras, and seaborn, the data is checked, prepped, accounted for bias by filtering for classifications with at least 1200 observations. It is then preprocessed by converitng the sequences into integers through a dictionary and padded so they are all of equal length. The to_categorical function is used to transform the class vector into a binary class matrix. The model is tuned with mlfow UI.
